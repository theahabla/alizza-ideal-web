# ブランド開発の１：コンセプトを決める

ブランドを開発するとき、まず初めにすることは、
ブランドのコンセプトの決定です。

よくある間違いは、誰にでも受けがよいように、
いろんなお弁当を作ってしまうこと。

つまり、お客さんにとって、他のお店と比べたときに、
あなたのお店のお弁当を選ぶ理由がなくなってしまうんです。

たしかに、何でも作れる人ってすごいです。

でも、やっぱりこの道一筋の職人のほうが、
美味しいお弁当を作れそうじゃないですか。

そっちの方を食べてみたいじゃないですか。

その人のお弁当を買っちゃうじゃないですか。

和食も、中華も、イタリアンも、フレンチも、
ピノイ風アジアン料理も、
全部合わせて１０年の経験がある料理人と、
和食だけで１０年やってきたという人、二人がいたとします。

どちらが美味しいお弁当を作れそうでしょうか？

単純に考えて、最初のいろんな料理ができる人は、
ひとつの料理に２年ずつ勉強した計算になります。

すると、和食に限って言えばその道１０年の人のお弁当のほうが美味しそうですよね。

たしかに、和食で１０年の人とピノイ風アジアン料理（なにこれ？）で勝負すれば、
最初の人のほうが美味し料理を作れそうでしょう。

でも、インターネットで販売すると、
アジアン料理の専門家とも比較されてしまうんです。

事実、昨年インターネットに進出した会社は、
競合会社が全国に広がってしまったために、
倒産してしまいました。
もともと業績は良かったのにです。
これは本当の話です。

ではまた、なにか進展があったら報告します。
