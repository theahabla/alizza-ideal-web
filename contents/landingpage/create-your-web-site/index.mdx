#### 副業は始めたい、でも何から手を付けていいのか

ただ、副業を始めようにも、次のようなことを不安に思っていませんか？

  - いったい、何を準備すればよいのか……
  - ネットショップを始めたいけど、何を売ったらいいんだろう
  - そもそも、どこから仕入れればいいかわからない
  - 商品の値段は、どうやって決めればいいのだろう？
  - 本当に、ずっと買ってもらえるのか不安だ

実は私は、３〜４年前くらいから、知人のお店の販売促進や、集客を手伝ってきました。
その後、副業で自分が作ったハンドメイド作品の販売を始めました。

そして、中国から値段の安い材料や、完成品を輸入して販売するようになりました。

２０２１年になってからは、海外在住の友人を通じて、輸出販売も手掛けたりしています。
今後越境ＥＣにも乗り出す予定です。

もちろん、はじめから簡単に売れることはありませんでした。

そこで、当初より商品を買ってもらうにはどうすればよいか、
勉強しながら続けています。

ここで大事なのは、ある心構えでした。

どんなことだったか、わかりますか？
非常に簡単なことです。

それは、上で書き分けたように、「売る」ではなく「買ってもらう」と考えることなンです。

「商品を売る」「お客さんに売る」ではなく、
「商品を買ってもらう」「お客さんが買ってくれる」と考えることです。

言い方の違いだけと思いますか？

このような考え方をするメリットについては、
Web サイト制作の付録、「ネット販売の基本の無料レクチャー」でお伝えします。

#### Webサイトを持ち、あなたと商品の魅力を伝える場所を作りましょう

独自 Web サイトを作ると、副業を始めるときに、こんなメリットがあります。

  - 実店舗がいらない
  - そのため、家賃や改装費などの開業費が安くすむ
  - 小さく始めて、段々とスケールアップしていける
  - 少しのミスなら、FX投資などと違って、致命的な失敗にはつながらない
  - Web サイトにアクセスした人の反応がすぐわかる
  - そのため、予測した反応ではなかったときに、軌道修正が簡単
  - 軌道修正を積み重ねていくことで、トヨタのようにカイゼンし発展していける

そして、Web サイトを使った副業がうまく回り始めると、こんなメリットがでてきます。

  - 生活に余裕ができた
  - バーチャルだけど、自分の夢だった素敵なお店を持てた
  - 自分が気に入ったものが、色んな人に好かれて買ってもらう喜びを味わえた
  - 副業を通して、スキルアップすることができた
  - 副業の経験を、本業の方に活かすことができた
  - 副業で得た知識や人脈を使って、会社の事業拡大に貢献できた
  - 将来会社になにかあっても、転職や独立を含めた選択肢が増え、仕事に余裕ができた
  
簡単に言えば、金銭的な面だけでなく、心にも、仕事にも余裕ができるようになります。

これまでの商品販売の経験や、ご商売をされている方々に聞いてわかったことがあります。
それは、小さな会社や個人にとって、独自サイトが非常に強力な武器になるということです。

以前までは、Amazon や楽天市場で販売したり、
ポータルサイトを使うのが良いというのが通説でした。

お客さんの目に付きやすいというのが理由です。

ただ多くの場合、そのような情報を発信しているのは、
ポータルサイトの関係者が多いことも事実です。

もしくはアフィリエイトで、
ポータルサイトを紹介しているブロガーなんかもそうですね。

##### 止められない独自 Web サイトのながれ

では、世界に目を向けてみましょう。

実は昨年頃から、米国を含む世界では、独自 Web サイトを作る人が増えてきています。

ビックリしたのですが、最近 Amazon は、
独自 Web サイトでネットショップを提供する BigCommerce 社を買収しました。

普通に考えて、Amazon 以外のネットショップが増えると、
ライバルが増えてしまいます。
つまり、独自 Webサイトが増えると、Amazon の損になってしまいます。

しかし Amazon は、独自 Webサイトを提供する側にまわったのです。
これは、それほどまでに独自 Webサイトは、
Amazon でも無視できないほどになったということです。

ネットで商品を販売するには、独自 Webサイトでネットショップを
開店するのが当たり前になったというわけです。

##### 副業が本業にもたらすメリット

さて、副業の経験を本業に活かすことができるというメリットも書きました。

繰り返しましょう。

  - 副業の経験を、本業の方に活かすことができた
  - 副業で得た知識や人脈を使って、会社の事業拡大に貢献できた
  - 将来会社になにかあっても、転職や独立を含めた選択肢が増え、仕事に余裕ができた

副業の経験をフィードバックすることで、本業の会社にも貢献することができます。
貢献することで、あなたは会社にとってなくてはならない社員になることもできます。

あなたは、会社に仕事をもらうためには仕事が選べない、卑屈な存在になりたいですか？
それとも、会社と対等な立場にたち、周りの人からも尊敬される存在になりたいですか？

副業は、あなたのような社員にとってのメリットばかりではありません。
あなたの会社にとってもメリットがあるンです。

もちろん、副業が忙しくて本業がおろそかになったり、
会社の機密情報を漏洩してしまっては本末転倒です。
そうならないように気をつけましょう。

でも、Web サイトを使えば、その危険は最小限に抑えられます。

特に、情報漏えいに関しては、WordPress を使うよりも、
